show a checkbox "Remind me"
when checked then
	show the input for the date and time picker
	show the date and time picker
	show the frequency dropdown list with "Once" selected by default
when unchecked then
	hide the input for the date and time picker
	hide the frequency dropdown list
	
the Task has the following fields related to the reminder:
-remindAt: Date
-reminderFrequency: string { Once, Daily, Weekly, Monthly, Yearly }
